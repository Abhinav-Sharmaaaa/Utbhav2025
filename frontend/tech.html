Technology Overview

How we detect safety risks in real-time

MediaPipe Pose

Detects 33 body keypoints for pose estimation

  • Identifies falls/slips through joint angles
  • Detects raised hands (help signals)
  • Monitors lack of movement

YOLO/YuNet Object Detection

Identifies dangerous objects and hazards

  • Weapons detection
  • Vehicle proximity alerts
  • Fire/smoke recognition
  • Animal detection

Safety Logic Engine

Combines inputs to assess danger levels

  • Pose + object proximity analysis
  • Environmental risk scoring
  • Real-time alert prioritization

System Architecture

Camera → Pose/Object Detection → Risk Analysis → Alert System
