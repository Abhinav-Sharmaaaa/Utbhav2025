Technology Overview

How we detect safety risks in real-time

YOLOv8 Pose Estimation

Detects 17 body keypoints for pose estimation.

  • Identifies falls by checking shoulder/hip alignment
  • Monitors for unusual poses or lack of movement
  • Provides real-time person tracking

YOLOv8 Object Detection

Identifies dangerous objects and hazards from 80 categories.

  • Dangerous object detection (knives, etc.)
  • Monitors for specific items in restricted areas
  • Provides location data for all detected items

Motion & Logic Engine

Combines inputs to assess danger levels in real-time.

  • Background Subtraction for motion analysis
  • Checks for overlap between motion and objects
  • Real-time alert prioritization and capture

System Architecture

Our processing pipeline is built for speed and accuracy.

Video Stream → Backend (Python/Flask)
    ↓
1. Frame Pre-processing (Resize)
2. Parallel Processing:
    - YOLOv8 Pose (Fall Detection)
    - YOLOv8 Object (Item Detection)
    - Background Subtractor (Motion)
3. Risk Analysis Engine
    (Is a (person) falling?)
    (Is a (danger object) moving?)
    ↓
4. Generate State (Threat/Notice/Clear)
    ↓
Frontend → (Stream, Logs, Captures)
